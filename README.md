## Tech Stack

**Client:** React, Tailwind, Ant Design

**State Management:** Zustand

## Run Locally

Clone the project

```bash
  git clone https://github.com/cmdanan/Pansmart.git
```

Go to the project directory

```bash
  cd Pansmart
```

Install dependencies using Yarn

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
